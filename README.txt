This Python program is a simple tool that allows you to download videos from YouTube directly to your computer. You can copy and paste the link of the video you are interested in, choose whether to download only the audio (in MP3 format) or the complete video (in MP4 format), and then select the folder on your computer where you want to save the downloaded file.

**What is it?**
- It is a program that facilitates the download of videos from YouTube.
- Created in Python, it uses libraries such as pytube to interact with the YouTube platform and tkinter for the graphical interface.

**What is it used for?**
- It is used to download videos from YouTube easily and quickly.
- It allows you to choose between downloading only the audio (in MP3 format) or the complete video (in MP4 format).

**What is used?**
- It uses a simple graphical interface where you can paste the link of the video.
- It utilizes the pytube library to interact with the YouTube API and download the videos.
- The tkinter library is used to create the application window and interface elements.
- The `filedialog` module of tkinter is also used to allow the user to select the destination folder for the download.
- Information and error messages are displayed using the `messagebox` library of tkinter.
